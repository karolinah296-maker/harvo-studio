# Harvo Studio

Statyczna strona internetowa Harvo Studio przygotowana do publikacji na Netlify.

## Publikacja na Netlify

1. Wrzuć projekt do repozytorium GitHub.
2. W Netlify wybierz `Add new site` -> `Import an existing project`.
3. Połącz repozytorium z GitHub.
4. Ustawienia zostaw domyślne:
   - Build command: puste
   - Publish directory: `.`
5. Po publikacji sprawdź formularz kontaktowy w zakładce `Forms`.

## Ważne pliki

- `index.html` - strona główna
- `oferta.html` - oferta i pakiety
- `portfolio.html` - portfolio
- `dziekuje.html` - strona po wysłaniu formularza
- `netlify.toml` - konfiguracja Netlify
- `_redirects` - krótkie adresy podstron
- `robots.txt`, `sitemap.xml`, `llms.txt` - SEO i indeksowanie
